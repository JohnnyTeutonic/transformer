Detailed Description

Manages the vocabulary for the tokenizer, including special tokens and mappings.

The Vocabulary class maintains bidirectional mappings between tokens (strings) and their corresponding IDs (integers). It handles:

  • Special tokens (PAD, UNK, BOS, EOS, MASK)
  • Basic vocabulary initialization
  • Token-to-ID and ID-to-token conversions
  • Noun identification for linguistic tasks
