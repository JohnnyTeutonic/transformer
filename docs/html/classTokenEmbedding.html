Detailed Description

Token embedding layer that converts token IDs to dense vectors.

The TokenEmbedding class manages the embedding lookup table that maps token IDs to continuous vector representations. Features include:

  • Learnable embedding weights
  • Gradient computation for training
  • CUDA acceleration support
  • Vocabulary projection for language modeling
