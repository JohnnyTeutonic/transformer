Detailed Description

Implementation of the Feed-Forward Network (FFN) used in transformer layers.

The Feed-Forward Network consists of two linear transformations with a ReLU activation in between, following the architecture: FFN(x) = max(0, xW₁ + b₁)W₂ + b₂

Features:

  • Two-layer neural network with ReLU activation
  • Dropout regularization
  • CUDA acceleration support
  • Gradient computation for training
