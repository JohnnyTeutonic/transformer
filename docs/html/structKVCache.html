Detailed Description

Key-Value cache for efficient autoregressive generation.

The KVCache struct implements caching of key and value tensors in transformer attention layers, enabling efficient autoregressive generation by avoiding redundant computation of keys and values for previously processed tokens. Features include:

  • Efficient memory management
  • Dynamic cache updates
  • Automatic size handling
  • Cache invalidation
