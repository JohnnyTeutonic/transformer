Detailed Description

Text tokenizer for converting between text and token sequences.

The Tokenizer class provides functionality for encoding text into token sequences and decoding token sequences back into text. It handles:

  • Special tokens (PAD, UNK, BOS, EOS, MASK)
  • Subword tokenization
  • Text preprocessing
  • Token caching for efficiency
