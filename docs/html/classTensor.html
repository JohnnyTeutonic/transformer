Detailed Description

A 4-dimensional tensor class for neural network computations.

The Tensor class provides a 4D data structure commonly used in deep learning, particularly for attention mechanisms and batch processing. It supports:

  • Matrix-like 2D operations
  • 4D tensor operations
  • Dimension permutation and transposition
  • Tensor multiplication
  • Conversion to/from matrices
