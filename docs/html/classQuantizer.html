Detailed Description

Post-training quantization for model compression.

The Quantizer class provides functionality for quantizing model weights and activations to reduced precision (e.g., 8-bit integers) after training. Features include:

  • Configurable bit width
  • Scale and zero-point calibration
  • CUDA acceleration support
  • Symmetric and asymmetric quantization
