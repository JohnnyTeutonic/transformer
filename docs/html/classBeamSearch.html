Detailed Description

Implements beam search decoding for sequence generation.

Beam search is a heuristic search algorithm that explores a graph by expanding the most promising nodes in a limited set. In the context of text generation, it maintains multiple hypotheses (beams) at each step and selects the most likely sequences. Features include:

  • Configurable beam width
  • Length penalty for balanced sequence lengths
  • Early stopping on EOS token
  • Score normalization

Performs beam search decoding.

Starting from initial logits, generates sequences by maintaining the top-k most likely hypotheses at each step. The search continues until either max_length is reached or all beams produce EOS token.
