Detailed Description

Thread-safe logging system for transformer operations.

The Logger class implements a singleton pattern to provide centralized logging functionality throughout the application. Features include:

  • File and console output
  • Error level distinction
  • Timestamp recording
  • Enable/disable control
  • Stream redirection
